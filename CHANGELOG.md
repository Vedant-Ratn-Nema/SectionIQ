# Changelog

## 0.1.0a1

- Renamed the public SDK to `SectionIQ` with `sectioniq` as the package namespace.
- Added compatibility shims for early `structured_pdf_rag` imports.
- Added public-domain Army TM corpus metadata and public benchmark queries.
- Added public corpus preparation and benchmark scripts.
- Improved table-row detection for aligned and collapsed technical PDF rows.
- Tightened warning/note detection to avoid false positives like component names.
- Added release safety checks for known confidential strings.
- Added GitHub CI and PyPI trusted-publishing workflow scaffolding.
